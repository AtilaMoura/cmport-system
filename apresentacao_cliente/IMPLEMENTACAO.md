# Implementação — Apresentação CMPort

## Arquivos necessários nesta pasta

```
apresentacao_cliente/
├── apresentacao.html      ← abrir no navegador para apresentar
├── logo_cmport.png        ← logo do leão (já existe)
└── IMPLEMENTACAO.md       ← este arquivo
```

## Como usar

1. Abra a pasta `apresentacao_cliente/`
2. Dê duplo clique em `apresentacao.html`
3. O navegador abre a apresentação em tela cheia automático

## Controles durante a apresentação

| Tecla | Ação |
|-------|------|
| `→` ou `Espaço` | Próximo slide |
| `←` | Slide anterior |
| `F` | Entrar/sair de tela cheia |
| `Home` | Primeiro slide |
| `End` | Último slide |
| Clique na tela | Próximo slide |

## Cores CMPort (para usar no PowerPoint se quiser)

| Cor | Hex |
|-----|-----|
| Azul escuro (fundo capa) | `#1A2E5A` |
| Azul royal (títulos) | `#1B5FBF` |
| Azul médio (ícones) | `#2E7DD4` |
| Dourado (destaques) | `#C8A95E` |
| Fundo claro | `#F0F4FA` |

## Estrutura dos 9 slides

1. **Capa** — identidade visual CMPort
2. **O Problema** — o que acontecia antes do sistema
3. **Visão Geral** — 8 módulos ativos
4. **Clientes e Notas Fiscais** — cadastro + importação automática
5. **Boletos e E-mail** — geração + envio automático
6. **Termos e Auvo** — PDF automático + integração de campo
7. **Dashboard** — gráficos e comparativos financeiros
8. **Antes × Depois** — tabela de valor entregue
9. **Encerramento** — próximos passos

## Requisitos

- Qualquer navegador moderno (Chrome, Edge, Firefox)
- Conexão com internet (para carregar as fontes Google)
- Sem instalação necessária
